Portfólio: Meus Trabalhos

Projeto 01: Design de Interação

Este projeto é um protótipo de interface bancária inspirado na estrutura do Banco Itaú. Com um design minimalista em tons de verde, o foco foi aplicar requisitos reais de um sistema financeiro, como a visualização clara de extratos e canais de atendimento ao cliente, priorizando uma navegação leve e intuitiva.

Projeto 02: Calculadora de Saúde e Bem-Estar (Ação Extensionista)

Desenvolvido como uma ação extensionista, este projeto foca na conscientização comunitária e educacional. Trata-se de uma ferramenta digital acessível que permite o cálculo de IMC, taxa calórica e hidratação diária, buscando promover hábitos saudáveis através de uma interface simples e informativa para o público geral.
